Guía Rápida de CSS3

Guía de propiedades, selectores, box model y layouts para dar estilo y vida visual a cualquier página web.

Sintaxis

Estructura de una Regla CSS

Una regla CSS se compone de un selector que apunta a un elemento HTML y un bloque de declaraciones con propiedades y sus valores.

/* Selector + bloque de declaraciones */
selector {
  propiedad: valor;
  otra-propiedad: valor;
}

/* Ejemplos */
h1 {
  color: #7c3aed;
  font-size: 2rem;
  text-align: center;
}

.card:hover {
  transform: translateY(-6px);
  box-shadow: 0 14px 36px rgba(0,0,0,.5);
}

Box Model

El Modelo de Caja

Cada elemento HTML es tratado como una caja rectangular compuesta por 4 capas concéntricas. Comprender este modelo es fundamental para controlar el espacio y los márgenes.

MARGIN — espacio exterior
BORDER — borde visible
PADDING — relleno interior
CONTENT — contenido real

Propiedades Clave

Propiedades Esenciales

color

Color del texto del elemento.

background

Fondo del elemento: color, gradiente o imagen.

font-size

Tamaño de la tipografía en px, rem o em.

display

Tipo de renderizado: block, flex, grid, none…

position

Posicionamiento: static, relative, absolute, fixed.

flex / grid

Sistemas de layout modernos y responsivos.

transition

Animaciones suaves al cambiar propiedades.

border-radius

Redondea las esquinas del elemento.
